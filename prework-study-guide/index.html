Prework Study Guide

HTML

  • HTML creates the structure of our webpage
  • Head element contains information about the webpage, helps the browser render the page.
  • The body element contains content visible to the user. Structured using semantic HTML elements.

CSS

  • CSS is used to style your webpage.
  • The most common way to add CSS to your page is by using an external stylesheet. You need to create a separate CSS file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Multiple elements can be styled using the same properties or class attributes can be used.

Git

  • Used to help developers with version control.
  • A main branch is created for a codeblock and comparaison or feature branches can be created when changes are made to the code.
  • Git commands can be used in the CLI:
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swithces to it
  • git add -A: stages changes to the code
  • A cheatsheet of Git commands can be found here: https://education.github.com/git-cheat-sheet-education.pdf
